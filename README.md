## 内容

- **首页** — 个人介绍 + 技能工具箱 + 项目经历 + GSAP 滚动动画
- **📝 技能分享** — 系列教程入口 + 随笔文章
  - 🤖 [AI 学习路径](https://cv-r.github.io/series/ai-learn) — 从 0 到 1 成为 AI Native 工程师（11 章）
  - 📗 [Vue 3 源码](https://cv-r.github.io/series/vue-source) — 逐行读懂核心源码（12 章）
  - ⚛️ [React 源码](https://cv-r.github.io/series/react-source) — 深入架构内核（13 章）

## 本地开发

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 构建到 dist/
```

## 部署

Push 到 `main` 分支 → GitHub Actions 自动构建部署到 GitHub Pages。
